= Conoce la virtualización de Red Hat OpenShift
:hardbreaks:
:allow-uri-read: 
:nofooter: 
:icons: font
:linkattrs: 
:imagesdir: ../media/


[role="lead"]
Red Hat® OpenShift® Virtualization permite a las organizaciones migrar, ejecutar y gestionar sin problemas máquinas virtuales (VM) tradicionales junto con cargas de trabajo modernas en una plataforma de aplicaciones unificada y de nube híbrida. OpenShift Virtualization está disponible como un operador incluido con cualquier suscripción a OpenShift. Este complemento de la OpenShift Container Platform ofrece una solución robusta para integrar máquinas virtuales con aplicaciones en contenedores. Utiliza Kernel-based Virtual Machine (KVM), el hipervisor de confianza del kernel de Linux, conocido por su sólido rendimiento, escalabilidad y estabilidad, para ofrecer virtualización de clase empresarial para máquinas virtuales como objetos nativos dentro de OpenShift.

image:redhat-openshift-044.png["Virtualización OpenShift"]



[[key-features-of-red-hat-openshift-virtualization]]
== Funciones clave de Red Hat OpenShift Virtualization

**Cargas de trabajo híbridas:** Migra y ejecuta máquinas virtuales con las ventajas propias de la nube, gestionándolas junto con contenedores y cargas de trabajo sin servidor mediante un conjunto de herramientas unificado.

**Modernización gradual:** Modernízate a tu propio ritmo adoptando contenedores, implementando microservicios o integrando IA, sin afectar las máquinas virtuales existentes.

**Integración con Kubernetes:** Aprovecha los recursos personalizados de Kubernetes para añadir capacidades de virtualización a tu clúster de OpenShift.



[[capabilities-of-red-hat-openshift-virtualization]]
== Capacidades de Red Hat OpenShift Virtualization

**Gestión de máquinas virtuales:** Crea y gestiona máquinas virtuales tanto de Linux como de Windows.

**Cargas de trabajo unificadas:** Ejecuta cargas de trabajo de pods y máquinas virtuales simultáneamente dentro de un clúster.

**Conectividad:** Accede a las máquinas virtuales a través de diversas consolas y herramientas de línea de comandos.

**Operaciones con máquinas virtuales:** Importar, clonar y migrar en tiempo real máquinas virtuales entre nodos.

**Gestión de recursos:** Maneja las interfaces de red y los discos de almacenamiento conectados a las máquinas virtuales.



[[compatibility]]
== Compatibilidad

Aunque la virtualización de OpenShift difiere conceptualmente de la de VMware vSphere, ofrece una plataforma moderna diseñada específicamente para administradores de virtualización con funcionalidades conocidas como: **Migración en vivo:** equivalente a vMotion en vSphere.

**Gestión del almacenamiento:** Las clases de almacenamiento en OpenShift se alinean con Storage Policy Based Management (SPBM) en vSphere.

**Almacenamiento persistente:** Las reclamaciones de volumen persistente (PVC) y los volúmenes persistentes (PV) se corresponden con los almacenes de datos en vSphere.
